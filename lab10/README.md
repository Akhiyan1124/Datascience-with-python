# Lab 10: Command-Line CSV Parser

## Overview
This project is a command-line CSV parser built using Python and argparse.

It allows users to:
- Parse CSV files
- Calculate statistics (average, minimum, maximum)
- Filter rows by column values
- Display filtered data
- Handle errors gracefully

---

## Features

✔ CSV parsing using built-in csv module  
✔ Statistical calculations  
✔ Row filtering  
✔ Command-line arguments with argparse  
✔ Error handling  
✔ Modular function-based structure
